kbdiffdi.features.feature
===========================

   Spatio-temporal data models  

**Classes**  

===============================================================  ===================================================================================== 
  Class                                                           Description    
===============================================================  =====================================================================================  
`feature.STFeatureStack() <./api/feature/STFeatureStack.html>`_  spatio-temporal data model for data at multiple timesteps
`feature.RasterStack() <./api/feature/RasterStack.html>`_        spatio-temporal raster data model for rasters at multiple timesteps
`feature.VectorStack() <./api/feature/VectorStack.html>`_        spatio-temporal vector data model for vector data at multiple timesteps
`feature.STFeature(...) <./api/feature/STFeature.html>`_         spatio-temporal data model for a data at a single point in time
`feature.Raster(...) <./api/feature/Raster.html>`_               spatio-temporal raster data model for a single raster at a single point in time
`feature.Vector(...) <./api/feature/Vector.html>`_               spatio-temporal vector data model for a single vector data at a single point in time
===============================================================  =====================================================================================
